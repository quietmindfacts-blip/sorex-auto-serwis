# Sorex Auto Serwis Wulkanizacja — Premium Automotive Workshop Website

## Local Development

```bash
npm install
npm run dev
```
Site will run at [http://localhost:5173](http://localhost:5173)

## Build For Production

```bash
npm run build
```

## Deploy

Deploy the `dist/` folder to Vercel or Netlify as a static site.

- Vercel: [vercel.com/new](https://vercel.com/new?utm_source=readme)
- Netlify: [app.netlify.com/drop](https://app.netlify.com/drop)
